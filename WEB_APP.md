# What Is This Web App?

**Fox Travel Egypt** is the official website for a travel company in **Hurghada, Egypt**. It is a booking-focused site where tourists can discover and book trips, sea excursions, and private transfers.

---

## In One Sentence

A mobile-first travel website that lets visitors **discover trips**, **book transfers**, and **contact the company** (including via WhatsApp), with clear pricing and simple flows.

---

## What It Does

| Area | Description |
|------|-------------|
| **Trips & tours** | Sea trips (snorkeling, diving, islands), desert safaris, city tours (e.g. Cairo, Luxor), day and multi-day tours. |
| **Transfers** | Airport ↔ hotel, city-to-city, and private transport. |
| **Contact** | WhatsApp and other contact options so users can ask questions or complete bookings. |

The site is built to **convert**: every section and page aims to help users book, pay, or get in touch—not just browse.

---

## Who It’s For

- **Tourists in Hurghada** (international, e.g. Europe, UK, Russia, Middle East).
- **Mobile-first** users who want quick info and simple booking.
- People who prefer **clear prices** and **easy WhatsApp** contact.

---

## Tech Snapshot

- **Next.js 16** (App Router) + **React 19** + **TypeScript**
- **Tailwind CSS v4** for styling; brand colors and typography in `app/globals.css`
- **TanStack React Query** + **Axios** for trips and other API data
- Trips data comes from a backend API; base URL is set via `NEXT_PUBLIC_API_URL`
